# دليل المستخدم العربي - GeoIndex Studio

## البداية السريعة

1. افتح الأداة.
2. اختر خريطة أساس. OpenStreetMap يعمل دون مفتاح.
3. ابحث عن مكان أو ارسم مضلعا لمنطقة الدراسة.
4. ارفع بيانات GeoJSON/KML/GPX/CSV/Shapefile ZIP أو GeoTIFF عند الحاجة.
5. اختر نوع التحليل ثم اضغط “تنفيذ التحليل”.
6. راجع لوحة “النتائج والمؤشرات”.
7. اضغط “تفسير النتيجة بالذكاء الاصطناعي”.
8. صدّر الخريطة أو البوستر أو البيانات.

## الرسم والقياس

يمكن رسم نقطة أو خط أو مضلع أو مستطيل. يحسب النظام المساحة والمحيط والطول باستخدام Turf.js. هذه القياسات تعليمية واستكشافية، وليست بديلا عن المسح الهندسي.

## رفع الملفات

الصيغ المدعومة:

- GeoJSON / JSON
- KML
- GPX
- CSV مع أعمدة `lat/lon` أو `latitude/longitude`
- Shapefile مضغوط ZIP
- GeoTIFF مموضع جغرافيا

عند رفع GeoTIFF متعدد الحزم، اضبط أرقام الحزم في لوحة التحليل. مثال شائع: Red = 3 وNIR = 4، لكن ذلك يختلف حسب المصدر.

## التحليلات

التحليلات داخل المتصفح تعمل فقط إذا توفرت البيانات اللازمة فعليا. خريطة الأساس لا تكفي لحساب NDVI أو NDBI. عند عدم توفر بيانات، تعرض الأداة السبب بدلا من إنتاج نتيجة وهمية.

## التفسير

إذا لم تدخل مفتاح AI، يستخدم النظام تفسيرًا داخليًا مجانيًا من مكتبة قواعد محلية. التفسير يبدأ من القيم المحسوبة ولا يضيف أرقاما غير موجودة.

## التصدير

يمكن تصدير:

- صورة PNG للخريطة والنتائج.
- بوستر GIS عمودي أو أفقي.
- PDF.
- GeoJSON/KML/Shapefile للبيانات Vector.

لا يتم تحويل Raster إلى Shapefile دون تحويل Vector حقيقي.
