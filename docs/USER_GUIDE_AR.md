# دليل المستخدم العربي

## الطريقة المختصرة

1. افتح تبويب البيانات والمصادر.
2. أدخل Google Cloud / Earth Engine Project ID وOAuth Web Client ID.
3. اضغط ربط Google Earth Engine.
4. ارجع إلى تبويب التحليل.
5. ابحث عن مدينة أو عاصمة واخترها من النتائج.
6. اختر السنة.
7. اختر المؤشر ثم اضغط تنفيذ التحليل.

لا يحتاج المستخدم إلى رسم نقطة أو مضلع، ولا رفع GeoJSON، ولا تحديد خادم GEE أو Asset ID.

## ماذا يحدث بعد اختيار المدينة؟

يحاول التطبيق استخدام حدود المدينة من نتيجة البحث. إذا لم يرجع محرك البحث حدودا واضحة، يستخدم التطبيق صندوقا مكانيا حول المدينة حتى يتمكن GEE من حساب المؤشر.

## المؤشرات

القائمة تعرض المؤشرات المباشرة التي يملك التطبيق وصفة جاهزة لها داخل Google Earth Engine، مثل NDVI وNDWI وNDBI وLST وNO2 وCHIRPS وVIIRS وESA WorldCover. عند اختيار مؤشر غير متوفر للسنة أو المنطقة سيظهر سبب الخطأ بوضوح.

## التفسير

زر التفسير يقرأ النتيجة الحالية فقط: اسم المؤشر، المدينة، السنة، المصدر، والقيم الإحصائية التي ظهرت. إذا غيرت المؤشر، سيطلب التطبيق حساب النتيجة الجديدة قبل تفسيرها حتى لا يخلط بين CO2 وNDVI أو غيرها.

## التصدير

يمكن تصدير الخريطة أو البوستر أو التقرير المتاح من تبويب التصدير. بعض طبقات GEE قد تمنع التقاط الصورة بسبب إعدادات CORS في المتصفح.
